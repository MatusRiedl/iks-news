# Iks News Discord Bot

This project is a Python Discord bot that posts a daily market news digest to a Discord channel at a configurable local time for each server.

The digest currently includes:

- a short daily overview, generated by Gemini when configured, falling back to Groq when Gemini fails, and then to local template logic otherwise
- macro headlines
- stock market headlines
- index snapshots
- top 5 gainers
- top 5 losers
- crypto headlines when they are selected by the AI layer or needed as overflow content

## What you need

- Python 3.11+
- a Discord bot token
- optionally a Gemini API key from Google AI Studio for primary AI ranking, deduplication, hard-news selection, overview writing, and Slovak translation
- optionally a Groq API key for AI fallback when Gemini is unavailable or over quota
- a machine or host that stays online when the post should run

## Setup

1. Create a virtual environment:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
pip install -e .
```

3. Create your env file:

```powershell
Copy-Item .env.example .env
```

4. Fill in:

- `DISCORD_TOKEN`
- optionally `GEMINI_API_KEY`
- optionally `GROQ_API_KEY`
- optionally `DEFAULT_TIMEZONE` such as `Europe/Budapest`
- optionally `DEFAULT_POST_TIME` in `HH:MM` 24-hour format
- optionally `DEFAULT_LANGUAGE` (`sk` by default, supports `en` and `sk`)
- optionally `DATA_DIR` if you want bot data somewhere other than `data`
- optionally `DISCORD_TEST_GUILD_ID` for instant slash command sync in one test server while keeping global commands enabled

5. Run the bot:

```powershell
python -m market_digest_bot
```

## Discord bot setup

1. Go to the Discord Developer Portal.
2. Create an application and add a bot user.
3. Under OAuth2 -> URL Generator:
   - select `bot`
   - select `applications.commands`
   - give it `View Channels`, `Send Messages`, and `Read Message History`
4. Invite the bot to your server with the generated URL.

For this repo, the bot itself only needs:

- `View Channels`
- `Send Messages`
- `Read Message History`

Notes:

- `applications.commands` is an OAuth scope, not a bot permission.
- The user installing the app into a server needs `Manage Server`.
- `/news_setup`, `/news_now`, and `/news_disable` default to `Manage Channels`, but a server admin can delegate them to a specific user or role in `Server Settings -> Integrations -> Iks News`.
- `/news_status` currently does not require special permissions.
- The bot does not require privileged intents for this project.

There is also a ready-made invite link file in the repo root:

- `Invite Iksperial News Bot.url`

This bot does not require the privileged `message_content` intent because it uses slash commands.

## Slash commands

- `/news_setup` to choose the target channel, post time, timezone, and digest language
- `/news_status` to view the current schedule
- `/news_now` to post the report immediately
- `/news_disable` to stop daily posting for a server

Example:

```text
/news_setup channel:#market-digest time_24h:08:30 timezone:America/New_York language:English
```

The time is interpreted in the timezone you choose.
Existing guilds stay on English until you explicitly change their language. New guilds inherit `DEFAULT_LANGUAGE`.

## How It Works

The digest is built in layers:

1. Curated direct RSS feeds are fetched first for macro, stock, and crypto coverage.
2. A hard-news filter removes obvious personal-finance advice, listicles, watch-live teasers, trading setups, and similar low-signal headlines.
3. If a category is still thin after filtering, Google News RSS is used as a preferred-source fallback.
4. The AI layer then tries providers in order:
   - Gemini first
   - Groq second
   - local non-AI fallback logic last

When AI is available, it deduplicates, ranks, writes the overview, and translates Slovak headlines. If both AI providers fail, the bot still posts a digest using the filtered headlines and a local overview, but the headlines remain in their original feed language.

## Notes

- If the bot restarts after the scheduled time and it has not posted yet for that local day, it will post on the next scheduler check.
- Global slash command sync can take a while. If you set `DISCORD_TEST_GUILD_ID`, the bot also syncs commands immediately to that guild for development while keeping global commands available everywhere else.
- `Run Iksperial News Bot.cmd` starts the bot locally.
- `Run Iksperial News Bot.cmd` and `Open Todays Digest.cmd` auto-install missing Python dependencies into the selected Python environment when needed.
- The bot only stays online while the local bot window is open and your PC stays awake and connected.

## Data sources

- Direct curated RSS feeds as the primary source:
  - CNBC Economy, Finance, and Investing
  - MarketWatch Top Stories and MarketPulse
  - Yahoo Finance RSS
  - CoinDesk, Cointelegraph, and Decrypt
- Google News RSS as an allowlist-filtered fallback when direct feeds are thin
- Yahoo Finance public screener data for top gainers and top losers
- Gemini 2.5 Flash as the primary optional AI layer for semantic deduplication, ranking, overview writing, and Slovak translation
- Groq as the optional secondary AI layer, currently using `openai/gpt-oss-120b`, when Gemini fails
- A hard-news filter that excludes obvious personal-finance advice, listicles, watch-live teasers, and trading-idea headlines from the digest before final selection
